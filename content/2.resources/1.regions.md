# Regions

Explore all available endpoints and data structures for regions.

## `Get all`

Method:  `GET`

```
/api/regions
```

Response:

```json
[
  {
    "id": 5,
    "name": "Qashqadaryo viloyati"
  },
  {
    "id": 6,
    "name": "Navoiy viloyati"
  }
]
```

---

## `Get by id`

Method:  `GET`

```
/api/regions/:id
```

Response:

```json
{
  "id": 5,
  "name": "Qashqadaryo viloyati"
}
```

---
